CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation and layout of HTML documents. It controls how elements like text, images, and layouts appear on a webpage, allowing developers to separate content (HTML) from design.

1.Inline CSS:Applied directly inside an HTML element using the style attribute.
2.Internal CSS:Written inside a style tag within the head tagsection of an HTML document.
3.External CSS:Written in a separate .css file and linked to the HTML document.

Basic CSS Properties

Property Description
color Sets the text color
background-color Sets the background color
font-size Sets the size of the text
text-align Aligns text (left, center, right)
